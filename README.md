# Tourism App | MAT Project

A **Java-based Android application** designed to simplify travel planning, from destination discovery to final booking. Developed as a group project at **ABV-IIITM**.

---

### 📋 Project Overview
The application provides an end-to-end user journey for travelers to browse cities, select accommodations, and book curated sightseeing packages. [cite: 7]

#### **Core Features**
* **Destination Selection**: Browse popular Indian cities such as Mumbai, Delhi-NCR, Bengaluru, and Kolkata. [cite: 37, 39, 40, 41, 46]
* **Interactive Hotel Discovery**: Compare stays via an interactive map interface with live price markers ranging from ₹2500 to ₹10,000. [cite: 59, 64, 70]
* **Tourist Attractions**: Explore landmarks like **City Palace** and **Hawa Mahal** with transparent pricing and guest ratings. [cite: 80, 81, 82]
* **Curated Packages**: Choose between single-day or multi-day guided sightseeing tours. [cite: 89, 96, 100]
* **Smart Checkout**: Integrated summary of stay details, add-ons (Breakfast/Spa), and final cost calculation including taxes. [cite: 107, 111, 114, 117]

---

### 🛠️ Tech Stack
* **Language**: Java
* **Platform**: Android Studio (Empty Views Activity)
* **Tools**: Google Maps SDK, XML Layouts [cite: 59]

---

### 🚀 Application Flow
1. **Home Page**: Search or select a popular destination. [cite: 14, 34]
2. **Category Selection**: Choose between **Hotels**, **Tourist Places**, or **Travel Packages**. [cite: 16, 18]
3. **Selection**: Filter and sort specific items based on budget and ratings. [cite: 61, 62]
4. **Checkout**: Review subtotal, taxes, and add-ons before confirming the booking. [cite: 111, 119]
