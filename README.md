# Tourism App | MAT Project

Note: The repository and application name are currently placeholders and are subject to change based on the final group decision.

A **Java-based Android application** designed to simplify travel planning, from destination discovery to final booking. Developed as a group project at **ABV-IIITM**.

---

### 📋 Project Overview
The application provides an end-to-end user journey for travelers to browse cities, select accommodations, and book curated sightseeing packages.

#### **Core Features**
* **Destination Selection**: Browse popular Indian cities, places, hotels and more.
* **Interactive Hotel Discovery**: Compare stays via an interactive map interface with live price markers. 
* **Tourist Attractions**: Explore landmarks with transparent pricing and guest ratings. 
* **Curated Packages**: Choose between single-day or multi-day guided sightseeing tours. 
* **Smart Checkout**: Integrated summary of stay details, add-ons, and final cost calculation including taxes. 

---

### 🛠️ Tech Stack
* **Language**: Java
* **Platform**: Android Studio 
* **Tools**: XML Layouts (subject to change)

---

### 🚀 Application Flow
1. **Home Page**: Search or select a popular destination. 
2. **Category Selection**: Choose between **Hotels**, **Tourist Places**, or **Travel Packages**. 
3. **Selection**: Filter and sort specific items based on budget and ratings. 
4. **Checkout**: Review subtotal, taxes, and add-ons before confirming the booking.
